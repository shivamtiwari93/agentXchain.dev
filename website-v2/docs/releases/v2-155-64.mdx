# AgentXchain v2.155.64

`v2.155.64` completes BUG-109 by making continuous full-auto mode run the supplemental checkpoint recovery itself when `--auto-checkpoint` is enabled. The previous patch identified the stranded accepted-turn dirt and routed operators to `checkpoint-turn`; this patch removes that operator step from the DOGFOOD-100 path.

## Bug Fixes

- **BUG-109: continuous auto-checkpoint recovery.** `agentxchain run --continuous --auto-checkpoint` now parses checkpoint-required assignment failures, runs `checkpointAcceptedTurn()` for the referenced accepted turn, and retries the same active run.
- **Visible recovery event.** Successful recovery emits `continuous_auto_checkpoint_recovered` with the session ID, checkpoint SHA, and recovered file metadata.
- **Fail-closed guard.** Continuous mode does not claim recovery when checkpointing returns `already_checkpointed` or `skipped`; unrecoverable dirty work remains visible as a blocker.
