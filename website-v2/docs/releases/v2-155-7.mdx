# AgentXchain v2.155.7

`v2.155.7` is a dogfood patch over `v2.155.6`. It fixes a PM idle-expansion
charter gap discovered during the tusq.dev dogfood: the charter told the PM
to produce a structured `idle_expansion_result` but did not specify the output
location (top-level JSON key or sidecar file) or the exact JSON schema. The PM
understood the concept but produced only text descriptions instead of the
required structured JSON object.

## Bug Fixes

- **Idle-expansion charter output format**: the PM idle-expansion charter now
  includes an explicit OUTPUT FORMAT section specifying both accepted output
  locations (top-level `idle_expansion_result` key in turn-result.json or
  sibling `idle-expansion-result.json` sidecar), the complete JSON schema for
  `new_intake_intent` and `vision_exhausted` branches, and an explicit
  instruction to produce the actual JSON object rather than describing it in
  text.
