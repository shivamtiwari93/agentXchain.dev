# AgentXchain v2.155.63

`v2.155.63` fixes BUG-109: if an accepted, already-checkpointed turn still owns dirty actor files named in its observed diff summary, AgentXchain now routes recovery through a supplemental checkpoint instead of blocking the next code-writing turn with generic commit/stash guidance.

## Bug Fixes

- **BUG-109: supplemental checkpoint recovery.** `detectPendingCheckpoint()` now identifies checkpointed latest accepted turns that still own dirty actor files from their observed diff summary.
- **Checkpoint the stranded files.** `checkpointAcceptedTurn()` can create a supplemental checkpoint, merge recovered files into history `files_changed`, and advance `accepted_integration_ref`.
- **Keep recovery narrow.** Dirty files are recoverable only when there are no active turns, the files are named in the accepted turn's observed diff summary, and the accepted artifact supports workspace/patch recovery.
