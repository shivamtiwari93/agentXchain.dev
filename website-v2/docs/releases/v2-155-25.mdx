# AgentXchain v2.155.25

`v2.155.25` surfaces Watch Mode results in the local operator dashboard and
adds comprehensive Watch Mode documentation.

## Features

- **Dashboard Watch view**: `agentxchain dashboard` now includes a `#watch`
  top-level view showing recent watch results with route status, delivery ID,
  event/intent/run detail, deduplication state, and error summaries.
- **Watch results API**: new `GET /api/watch-results?limit=<n>` endpoint on the
  dashboard bridge server returns the most recent watch results as JSON.
- **Dashboard invalidation**: the file watcher monitors
  `.agentxchain/watch-results/` for new or changed result files and pushes
  live updates to connected dashboard clients.
- **Watch Mode docs page**: full operator-facing documentation at
  `/docs/watch-mode` covering HTTP listener setup, signed and unsigned curl
  recipes, CLI reference, endpoint contract, event routing configuration,
  supported event categories, security guidance, and three complete config
  examples (automated PR review, CI failure repair, manual issue triage).
