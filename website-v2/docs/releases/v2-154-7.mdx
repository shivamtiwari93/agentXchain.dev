# AgentXchain v2.154.7

`v2.154.7` is the BUG-52 third-variant discriminator repair release. It ships
the realistic PM `needs_human` handoff fix that `v2.154.5` missed, while
preserving verification-gated QA safety for synthetic standing-gate recovery.

## Bug Fixes

- **BUG-52 realistic human-signoff handoff**: `agentxchain unblock <hesc>` now
  advances a standing pending human-approval gate even when the accepted PM turn
  used `status: "needs_human"`, left `phase_transition_request: null`, and set
  `proposed_next_role: "human"`. The discriminator requires that the accepted
  turn contributed at least one of the gate's required files and that all
  required files exist on disk.

- **BUG-52 verification-gated synthetic source safety**: synthetic standing-gate
  transition sources no longer fabricate verification for gates that declare
  `requires_verification_pass: true`. If the latest blocked turn did not pass
  verification, `unblock` resumes in the current phase instead of advancing.

- **BUG-52 spec and decision alignment**:
  `.planning/BUG_52_RECONCILE_SOURCE_RESOLUTION_SPEC.md` and
  `.planning/DECISIONS.md` now record the artifact-contribution discriminator
  and the verification-preserving synthetic-source guard.
