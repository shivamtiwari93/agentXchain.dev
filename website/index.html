The CLI

9 commands. Everything you need.

Install globally or run with npx. No accounts, no platform, no setup beyond npm install.

agentxchain init
Create a project folder. Pick a team template or define custom agents.
agentxchain start
Launch agents in Cursor, Claude Code, or VS Code via API.
agentxchain watch
The referee. Coordinates turns, enforces TTL, wakes agents automatically.
agentxchain status
Lock holder, phase, turn number, Cursor agent status — at a glance.
agentxchain claim
Human takes control. Pauses all Cursor agents instantly.
agentxchain release
Hand the lock back. Wakes the next agent automatically.
agentxchain stop
Terminate all running agents via API. Clean shutdown.
agentxchain config
Add/remove agents, change rules, update settings — without editing JSON.
Features

Protocol v3: built for real use

Not a toy protocol. Built-in safeguards for the failure modes that happen when AI agents actually run.

Claim-based turns

No fixed order. Agents claim the lock when free. The watch process decides who goes next.

Lock TTL

Stale locks auto-released after timeout. No more deadlocks from crashed or hung agents.

Verify command

Agents must pass npm test (or your command) before releasing. No broken code handed off.

Human-in-the-loop

agentxchain claim pauses all agents. Fix things yourself. release hands back.

State / history split

state.md (overwritten) + history.jsonl (append-only). Agents read minimal context per turn.

Team templates

SaaS MVP, Landing Page, Bug Squad, API Builder, Refactor Team. Or define your own.
