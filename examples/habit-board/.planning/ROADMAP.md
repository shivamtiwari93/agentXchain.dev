# Habit Board — Roadmap

## Goal

Ship a minimal but complete consumer habit tracker that proves AgentXchain can govern web application delivery with a designer-in-the-loop workflow.

## Delivery Slices

1. **Slice 1 — Core API + persistence**: CRUD habits, check/uncheck, JSON file storage
2. **Slice 2 — Streak logic**: current streak, longest streak, computed on read
3. **Slice 3 — Frontend**: responsive card UI, add/delete/toggle, streak display
4. **Slice 4 — QA**: acceptance matrix, edge case coverage, ship verdict

## Non-Goals

- Authentication or multi-user support
- Mobile native app
- Cloud deployment or hosting
- Analytics or habit insights beyond streaks
