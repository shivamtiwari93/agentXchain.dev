# Operator Jobs

## Team Admin

Operators must be able to create teams with a delivery channel, reminder hour, timezone, and retention setting, then add members without editing files manually.

## Standup Collection

Operators need one place to capture async standups and re-submit them if plans change later in the day.

## Failure Recovery

If members miss standup or the backlog grows, operators need a reminder preview and a retention/prune path that keeps the data set manageable.
